# 🌍 ESG Intelligence Dashboard

A full-stack ESG (Environmental, Social, Governance) analytics dashboard built using Django, React, REST APIs, and CSV ingestion workflows.

This platform simulates enterprise ESG analyst operations including:
- CSV data ingestion
- Emission normalization
- Analyst review workflow
- Approval & flagging system
- ESG monitoring dashboard

---

# 🚀 Features

## Backend (Django + DRF)
- REST APIs
- CSV ingestion
- SQLite database
- ESG emission record management
- Analyst workflow APIs

## Frontend (React)
- Premium dashboard UI
- Live API integration
- Upload workflows
- Approval & flagging actions
- Dynamic analytics cards

---

# 📊 ESG Workflow

1. Upload emission CSV
2. Backend processes records
3. Dashboard displays ESG data
4. Analyst reviews records
5. Records can be:
   - Approved
   - Flagged
